# Gautier — Les Cruautés de l’Amour (1879)

Manual relationship annotation according to the current project codebook.

Source file used locally: `gautier_cruautesamour (1).txt`.

The Excel workbook in this folder contains the relation-level annotation table for the volume, treated as a collection of internal stories.

A fuller local copy with additional `Codebook`, `JSON`, and `Notes` sheets was also generated during the run; the Git-tracked `.xlsx` keeps the annotation table itself to minimize binary-file size.
