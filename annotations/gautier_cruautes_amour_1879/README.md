# Gautier — Les Cruautés de l’Amour (1879)

Manual relationship annotation according to the current project codebook.

Source file used locally: `gautier_cruautesamour (1).txt`.

The Excel workbook in this folder contains:

- `Annotations`: relation-level annotations;
- `Codebook`: allowed labels and compact definitions;
- `JSON`: schema-compatible annotation object;
- `Notes`: provenance metadata and input hash.
