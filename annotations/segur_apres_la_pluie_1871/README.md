# Comtesse de Ségur — Après la pluie, le beau temps

This folder contains a manual draft annotation workbook for the uploaded text file `segur_pluie.txt`.

Workbook:

- `segur_pluie_annotation_relationships.xlsx`

Contents:

- `Annotations`: 3 retained relationship annotations.

Status: draft manual annotation for review before use as gold data.
